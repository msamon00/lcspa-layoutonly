# LCSPA Layout Website

This repository contains the official website project for LCSPA (working layout version). The site is currently in active development and serves as the foundational structure for the business’s future full online presence.

The project is hosted and version-controlled using GitHub.

## 🌐 Live Site

Once GitHub Pages is enabled, the site will be available at:

```
[https://msamon00.github.io/lcspa-layoutonly/]
```

## 📁 Current Features

* Basic website layout structure
* Homepage design and styling
* Responsive front-end framework (mobile-friendly in progress)
* Organized file structure for scalable development
* Git-based version control for updates
